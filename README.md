## üõ†Ô∏è Getting Started

To run this project locally:

1. **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    # or
    bun install
    ```

2. **Start the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    # or
    bun dev
    ```

3. **Open your browser:**  
   Visit [http://localhost:3000](http://localhost:3000) to view the app.
